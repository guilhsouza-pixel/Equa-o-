# Calculadora de Equação do 2º Grau

Site simples em HTML, CSS e JavaScript para resolver equações do tipo:

```text
ax² + bx + c = 0
```

## Funcionalidades

- Calcula o discriminante (delta)
- Calcula as raízes reais quando existirem
- Informa quando não há raízes reais
- Calcula o vértice da parábola
- Mostra a concavidade
- Gera o gráfico da função quadrática em Canvas

## Como usar

Abra o arquivo `index.html` no navegador ou publique o projeto na Vercel.

## Publicação na Vercel

Configuração recomendada:

- Framework Preset: Other
- Build Command: vazio
- Output Directory: `.`
- Install Command: vazio

Depois clique em Deploy ou Redeploy.
